HTML Hyper Text Markeup Language

웹문서의 DOM(문서구조)를 설계하는 시멘틱 마크업 언어를 의미한다

html문서의 특징

  • 요소(element), 태그(tag), 속성(attribute)이 존재한다
  • Node(parent,children,decendent,siblings)가 존재한다
  • Node는 정해진 구조를 준수해야 한다

시멘틱요소

구획요소

  • main : 웹문서의 주요내용
  • section,article : 제목과 내용을 포함하는 주제를 출력하는 영역
  • header : 사이트의 정체성을 출력하는 영역
  • footer : 사이트의 보조적인 정보를 출력하는 영역
  • aside : 사이트의 부수적인 정보를 출력하는 영역
  • nav : 메뉴기능을 제공하는 영역
  • div : 레이아웃 영역

다양한 논리 요소

  • em : 중요함
  • strong, mark : 강조
  • small : 보조적의미
  • i : 구분, 다름의 목적
  • blockquote : 인용
  • cite : 인용처
  • time : 시간표시
  • address : 주소, 전화 등..
  • span : 의미가 없음
  • abbr : 약어
  • dfn : 용어

물리적 요소

  • b : 두꺼운글자
  • u : 밑줄
  • strike : 취소선
  • sup : 윗첨자
  • sub : 아래첨자
