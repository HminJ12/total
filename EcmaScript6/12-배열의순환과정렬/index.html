1. indexOf

  • 배열안에서 특정값이 몇 번째 인덱스에 있는지를 알아낸다
  • 값이 존재하지 않으면 -1을 반환
  • 동일한 값이 여러개 있더라도 첫번째 인덱스만 반환 후 종료
      ['가','나','다']

2. findIndex

  • 배열 값으로 객체나 배열이 중첩되어 있을 경우 인덱스 검색
  • 콜백함수 내에서의 매개변수와 조건문과 리턴문을 사용
      [
        {'이름':'홍길동','나이':'24'},
        {'이름':'아무개','나이':'22'},
        {'이름':'무명씨','나이':'23'},
      ]

3. findIndex

  • 배열 값으로 객체나 배열이 중첩되어 있을 경우 인덱스 검색
  • 콜백함수 내에서의 매개변수와 조건문과 리턴문을 사용
      [
        {'이름':'홍길동','나이':'24'},
        {'이름':'아무개','나이':'22'},
        {'이름':'무명씨','나이':'23'},
      ]

4. forEach

  • 배열의 길이만큼 순환
  • 콜백함수 내에서의 매개변수와 조건문과 리턴문을 사용
      [
        {'이름':'홍길동','나이':'24','성별':'남'},
        {'이름':'아무개','나이':'22','성별':'여'},
        {'이름':'무명씨','나이':'27','성별':'남'},
        {'이름':'영희','나이':'20','성별':'여'},
        {'이름':'순희','나이':'26','성별':'여'},
      ]
