배열

  • 배열의 선언방식과 호출 
            let arr = ['가','나','다']
        arr[0] // '가'
  • array.length : 배열의 길이 
            let arr = ['가','나','다']
        arr.length // 3
  • 배열의 값 추가, 값 삭제 
          let arr = ['가','나','다']
      arr.push('라') //['가','나','다','라'] 
      배열의 뒷쪽에 값을 추가 
    
      arr.unshift('A') //['A','가','나','다'] 
      배열의 앞쪽에 값을 추가
      arr.pop() //['가','나']
      arr.shift() //['나','다']
  • 배열의 특정 인덱스를 기준으로 값 추가, 삭제 
            let arr = [0,1,2,3]
        arr.splice(2,0,'가') // [0,1,'가',2,3]
        arr.splice(2,2) //[0,1,3] 위에 '가'를 넣은 기준으로
  • 문자열을 배열로 전환 split 
            let str = '안녕'
        str.split() //['안녕'] 배열 통으로 가져온다
        str.split('') // ['안','녕']
        let str = '010-1234-5678'
        str.split('-') //['010','1234','5678']
  • 배열을 문자열로 전환 join 
            let arr = ['홍','길','동']
        arr.join('-') // '홍-길-동'
        arr.join('') // '홍길동'
        arr.join() // '홍,길,동'

1. push, unshift, pop, shift(배열의 추가 및 삭제)

2. splice (배열의 특정 인덱스를 기준으로 추가나 삭제)

3. split (문자열을 배열로 전환)
